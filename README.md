# Kickstart-Nvim

A simple, minimal, modular neovim config inspired by [kickstart-nvim](https://github.com/nvim-lua/kickstart.nvim).

---

## Installation

```bash
git clone --depth=1 https://github.com/amritxyz/kickstart-nvim.git ~/.config/nvim
nvim
```
